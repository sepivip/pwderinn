## Deploy to GitHub Pages

```bash
# Initialize git (if not already)
git init

# Add all files
git add .

# Commit
git commit -m "Initial commit: Powder Inn clone"

# Create GitHub repo and push
git remote add origin https://github.com/YOUR-USERNAME/powderinn.git
git branch -M main
git push -u origin main

# Enable GitHub Pages in repository settings
```

Your site will be live at: `https://YOUR-USERNAME.github.io/powderinn/`
